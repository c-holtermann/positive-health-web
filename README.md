# positive-health-web

LaTeX / Tikz spiderweb for positive health

Adapted from https://texample.net/tikz/examples/spiderweb-diagram/

Conversion to svg:
``` bash
inkscape --without-gui --file=web_only.pdf --export-plain-svg=web_only.svg
```
